---
title: 'Generate Presentation'
api: 'POST /generate_presentation'
description: 'Generate a presentation based on a user prompt'
---

The `generate_presentation` endpoint allows you to create a presentation by providing a text prompt. The API will process your request and return a download URL for the generated presentation.

## Request

<ParamField query="user_prompt" type="string" required>
  The text prompt describing the presentation you want to generate. This should include details about the content, style, and any specific requirements for the presentation.
</ParamField>

## Response

The API returns a JSON object containing a download URL for the generated presentation.

<ResponseField name="download_url" type="string">
  A URL where the generated presentation can be downloaded. This URL is temporary and should be used promptly.
</ResponseField>

## Example

```bash
curl -X POST https://api.aidocmaker.com/generate_presentation \
  -H "Content-Type: application/json" \
  -d '{
    "user_prompt": "Create a presentation about the benefits of artificial intelligence in healthcare, including 5 slides with statistics and examples"
  }'
```

### Response

```json
{
  "download_url": "https://api.aidocmaker.com/download/presentation/abc123"
}
```

## Error Handling

The API may return the following error responses:

<ResponseField name="error" type="string">
  A description of the error that occurred
</ResponseField>

Common error codes:
- `400 Bad Request`: Invalid or missing parameters
- `401 Unauthorized`: Invalid or missing authentication
- `429 Too Many Requests`: Rate limit exceeded
- `500 Internal Server Error`: Server-side error
